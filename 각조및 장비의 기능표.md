# 2. 각조 및 장비의 기능 (표 형식 정리)

## 2-1. 각 조의 기능

================================================================================
  순서 | 조 명칭              | 역할                           | 유지관리 포인트
--------------------------------------------------------------------------------
   1  | 폭기조               | - 혼합액농도(MLSS) 유지           | - 1~4조 폭기 정상 확인
      | (Aeration Tank)     | - 1차 처리수 유기물질을            | - DO 센서 수시 확인
      |                     |   미생물로 호기성 산화 제거         | - 반송펌프 작동 확인
      |                     |                               | - DO 센서 세척 (주 1회)
      |                     |                               | - 터보 브로워 필터 교체 (주 1회)
      |                     |                               | - 반송펌프 압력/토출량 체크
--------------------------------------------------------------------------------
   2  | 분리막조              | - 침지형 분리막으로                | - 차압계 확인 (1회/일)
      | (Membrane Tank)     |   고농도 MLSS 유지               | - 흡입펌프 작동 확인
      |                     | - 미생물로 영양염류 및             | - 수위센서 작동 확인
      |                     |   유기물질 산화 제거               | - 포기 정상 확인
      |                     |                               | - 흡인펌프 순간유량계/차압계 확인
      |                     |                               | - 레벨스위치 청소 (2회/월)
      |                     |                               | - 주간세정 (주 1회)
      |                     |                               | - 인라인세정 (월 1회)
--------------------------------------------------------------------------------
   3  | 1차 처리수조           | - 가압부상조 거친 1차 처리수        | - 폐수 색상/냄새 확인
      | (1st Treatment Tank)| - 수리부하량 및 오염부하량          | - 반응조 이송 원활성 확인
      |                     |   균등화                        |
      |                     | - 일정시간 저류 후                |
      |                     |   펌프로 반응조 이송               |
--------------------------------------------------------------------------------
   4  | 반응조/               | - 유입 폐수를 PAC로 응결           | - 약품투입량 점검
      | PH조정조/             | - 가성소다(NaOH)로 중화반응         | - 약품펌프 이송능력 점검
      | 응집조                | - 폴리머로 최종 응집               | - PH 센서 세척/정상작동 확인
      |                     | - 생성 플록(Floc)을               | - 포터블 측정/리트머스종이 확인
      |                     |   가압부상조로 자연유하              |
--------------------------------------------------------------------------------
   5  | 가압부상조             | - 플록(Floc) 가압부상              | - 운전가동 상태 점검
      | (DAF System)        | - 스컴 및 처리수로 고액분리           | - 고액분리 상태 점검
      |                     | - 처리수를 1차 처리수조로             | - Scum 상태 점검
      |                     |   이송                           | - 에어 상태 확인
      |                     |                                 | - 순환수 가압펌프 압력 확인 (1회/일)
      |                     |                                 | - 공기용해접촉장치 압력 확인
--------------------------------------------------------------------------------
   6  | 여과수조              | - 분리막 통과한 처리수               | - 여과수조 이송펌프 A/B
      | (Filtrate Tank)     |   임시 저류                       |   토출압력 확인
      |                     | - 펌프로 A/C FILTER 또는           | - 펌프 관리/배관 개폐 확인
      |                     |   방류수조로 이송                   |
--------------------------------------------------------------------------------
   7  | 여액조                | - 가압부상조 SCUM 및               | - 여액조 펌프 유량/토출압력 확인
      | (Filtrate Return    |   슬러지 인발 탈수 여액              | - 배관 막힘 여부 확인 (육안)
      | Tank)               | - 탈수여액 임시 저류                 | - 펌프 유량 관리
      |                     | - 여액펌프로 반응조 재이송             |
--------------------------------------------------------------------------------
   8  | 슬러지 저류조          | - 가압부상조 SCUM 및                | - 적정 수위 유지
      | (Sludge Storage     |   슬러지 인발 발생                  | - MLSS 증가에 따른 인발량 증가
      | Tank)               |   슬러지 탈수 전 임시 보관            |   대응
      |                     |                                  | - 슬러지 이송펌프 유량 조절
--------------------------------------------------------------------------------
   9  | A/C FILTER          | - MBR 여과수 수질 악화 시             | - 미가동 시 물 배수 (동파방지)
      | (Activated Carbon   |   비상대응                          | - 밸브 조작 운영 숙지
      | Filter)             | - 활성탄으로 흡착/여과                 | - 차압 상승 시 역세 실시
      |                     | - 오염물질 저감                      | - 역세수 육안체크 (활성탄/여과사
      |                     |                                   |   유출 여부)
--------------------------------------------------------------------------------
  10  | 방류조               | - A/C FILTER 및 여과수조              | - 방류수 이송펌프 토출압/유량 확인
      | (Discharge Tank)    |   거친 처리수 임시 저류                 | - 유량계 정상작동 확인 (1회/일)
      |                     | - 펌프로 방류                         | - 판넬 화면 유량과 현장 유량계
      |                     |                                    |   지침 일치 확인
================================================================================

## 주요 운전 인터록 요약

================================================================================
  조건                           | 연동 동작
--------------------------------------------------------------------------------
  유량조정조 펌프 ON             | → DAF SYSTEM 자동 ON
                                 |   (SOL밸브→순환펌프→약품펌프→교반기)
--------------------------------------------------------------------------------
  1차처리수조 HH                 | → 유량조정조 펌프 OFF
--------------------------------------------------------------------------------
  분리막조 HH                    | → 1차처리수조 펌프 OFF
--------------------------------------------------------------------------------
  분리막조 HHAL                  | → 고수위 경보 + 1차처리수조 펌프 OFF
--------------------------------------------------------------------------------
  여과수조 H                     | → 흡입펌프 정지
--------------------------------------------------------------------------------
  방류조 HH                      | → 여과수조 펌프 OFF
--------------------------------------------------------------------------------
  PH 센서 High                   | → NaOH 펌프 정지
--------------------------------------------------------------------------------
  과전류 발생                    | → A-B 펌프 자동 대치 가동
--------------------------------------------------------------------------------
  흡입펌프 7분 초과 가동         | → 자동 정지 (분리막 폐쇄 방지)
================================================================================

## 유지관리 핵심 포인트

### 일일 점검 (1회/일)
- 차압계 확인 (분리막 상태)
- 각 조 수위/색상/냄새 확인
- 순환수 가압펌프 압력 확인
- 공기용해접촉장치 압력 확인
- 방류 유량계 정상작동 확인

### 주간 점검 (1회/주)
- DO 센서 세척
- PH 센서 세척 및 실측 대조 (포터블/리트머스)
- 터보 브로워 필터 교체
- 터보 브로워 A/B 교번 운전
- 주간세정 실시 (분리막)

### 월간 점검 (2회/월)
- 분리막조 레벨센서 인양 청소

### 월간 점검 (1회/월)
- 인라인세정 실시 (분리막)

### 필요 시 점검
- 차압 회복 안 될 시 → 계외세정 실시
- 약품 잔량 확인 및 보충
- 펌프 토출압/유량 이상 시 점검
